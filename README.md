
# Aura Reservation System 

# Aura Reservation System is a PHP and MySQL-based reservation management platform designed to streamline the management of office spaces and desks.
# 🚀 Features

    User Profile: Allows users to manage their personal information.

    Make a Reservation: Book office spaces / desks for specific dates and times.

    Reservation Management: View and cancel existing reservations easily.

    Database Integration: All operations are synchronized in real-time with a MySQL database.

# 🛠️ Technologies Used

    Backend: PHP

    Database: MySQL (MariaDB)

    Frontend: HTML, CSS, JavaScript (Bootstrap)

    Server: XAMPP / Apache
